DEB package v.4.1.3

DEB package 64-bit v.4.1.3

DEB package 32-bit v.4.1.3

Install this package and launch BORIS from Education menu.

On Debian, with dpkg. Open a terminal as root and type:

dpkg -i boris_##VER##-0.deb
apt-get -f install

On Ubuntu, open with the "Ubuntu Software Center" and install it (sudo password required)
