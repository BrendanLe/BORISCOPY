Features

  • Coding and describing visual and audio information using an integrated playback
  • Configuration of an unlimited number of behaviors that can be organized in categories
  • Configuration of an unlimited number of modifiers for each behavior. The modifiers can be of 3 type (single selection, multiple selection and numeric) and can be organized in an unlimited number of sets.
